# Aplicação react 

#### Como usar este projeto com docker 

`$ git clone `

`$ cd react-app-igti`

`$ docker build -t react-app-igti . `

`$ docker run -p 3000:3000 react-app-igti -d`

` Aplicação disponível em http://localhost:3000`


#### Como usar este projeto sem docker 

`$ git clone `

`$ cd react-app-igti`

`$ npm install`

`$ npm start`

` Aplicação disponível em http://localhost:3000`
